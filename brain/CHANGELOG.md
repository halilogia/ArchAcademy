# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-19

### Added
- **Core Architecture Engine**: Implementation of Clean Architecture, DDD, and Vertical Slice interactive guides.
- **Architect's Compass**: Added the requirement-based architectural wizard.
- **Master Matrix**: High-performance comparison tool for different architectural styles.
- **Architect's Glossary**: Interactive dictionary with 50+ architectural terms and Guru Tips.
- **Progress Tracking**: Local-storage based progress system to track user's learning journey.
- **Architect's Journey**: Career roadmap visualization (Craftsman to Visionary).
- **Glassmorphic UI**: Premium design system implemented with Vanilla CSS and Framer Motion.
- **GitHub Infrastructure**: Issue templates, PR templates, and CI/CD workflows for community collaboration.

### Changed
- Refactored `Vibe Coding` module into a dedicated `Code Vibe Laboratory` for better educational impact.

### Fixed
- Fixed HMR (Hot Module Replacement) issues in large glossary data files.
- Corrected various JSX closing tag errors in nested animation components.
